Help & Tutorials

Getting Started

Learn the basics of using MyShapeCreator and navigate through the drawing tools.

  • Choose your drawing tool from the left panel
  • Select colors and brush sizes
  • Start creating on the canvas
  • Save your work to the gallery

Drawing Techniques

Master the art of digital shape creation with these essential techniques.

  • Use the grid for precise measurements
  • Hold shift while drawing lines for straight edges
  • Practice basic shapes before complex designs
  • Use different brush sizes for details

Educational Goals

Understand how MyShapeCreator supports learning objectives in STEM education.

  • Spatial reasoning development
  • Geometric principle understanding
  • Pattern recognition skills
  • Mathematical visualization

TEASync Integration

Advanced users can integrate with TEASync framework for collaborative learning.

  • Team-based drawing projects
  • Teacher dashboard integration
  • Progress sharing capabilities
  • Curriculum alignment tools
